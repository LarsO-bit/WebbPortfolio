# About the Project

This project is a personal web portfolio created as part of the Web Development Frontend course. The purpose of the project is to present myself, my skills, and my projects on a professional website.

The design is inspired by Synthwave / Retrowave aesthetics, featuring neon colors, a dark background, and a futuristic style influenced by 1980s retrofuturism.

The portfolio is built using HTML5, CSS, and JavaScript without any frameworks in order to demonstrate an understanding of fundamental frontend development concepts.

## Features

The website includes:

- Portrait image
- Downloadable CV button
- Navigation menu (Navbar)
- Links to social media and GitHub
- About Me section
- Skills section displaying the technologies I can work with
- Portfolio section with projects (currently placeholders)
- References / Testimonials
- Contact form
- Footer

The website is also responsive and adapts its layout for both mobile and desktop devices.

## Technologies

- HTML5
- CSS3
- JavaScript
- Flexbox / Grid
- Fetch API
- Git & GitHub

## Structure and Solutions

The navigation bar is loaded dynamically from a separate HTML file using JavaScript. This keeps the code more organized and follows the DRY principle, since the navbar only needs to be updated in one place.

The website also includes a simple music player that allows the user to play and switch between multiple songs.

## Future Improvements

During development I also experimented with implementing a mouse trail effect where neon circles follow the mouse cursor. While the effect worked, it proved to be relatively resource-intensive, so it is currently not included in the final version.

It may be optimized and implemented in the future.
